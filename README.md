# Summer Trip 2026

A simple, view-only family vacation calendar for **August–September 2026**, published as a static GitHub Pages site.

- **Live page:** enabled via GitHub Pages (main branch, root).
- **What it shows:** two color-coded month calendars (August & September 2026), a day-by-day plan, and a generic family packing checklist.
- **No login, no tracking, no JavaScript.** The family just opens the link and refreshes to see the latest version.

## Updating

Everything lives in a single self-contained [`index.html`](index.html) (all CSS inline, no external fonts or scripts). To change plans, edit `index.html` and commit to `main` — GitHub Pages redeploys automatically. Refresh the page to see updates.
